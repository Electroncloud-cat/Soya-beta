📚 帮助文档

涟宗也 v0.2 - 功能说明、变量参考与更新日志
💬 基础对话功能

与涟宗也进行自然对话，支持文本输入和多种交互方式。

  • 输入框支持 Shift+Enter 换行，Enter 发送
  • 消息支持多行输入，每行会显示为独立的聊天气泡
  • AI 回复会自动解析动作指令（如 [MOTION:wave]）并触发 Live2D 动作
  • 支持引用回复：点击消息旁的引用按钮，可以引用该消息进行回复
🧠 记忆系统

涟宗也拥有长期记忆能力，会主动记住对话中的重要信息。

  • 自动记录用户偏好、重要事件、情感变化等信息
  • 记忆分为三种类型：dynamic（可变）、permanent（永久）、feel（感受）
  • 每条记忆包含重要度、情感效价、唤醒度等维度
  • 在设置中可以查看、编辑和删除记忆
  • 记忆会自动参与对话上下文，让 AI 更了解你
💖 情感系统

涟宗也具有动态情感状态，会根据对话内容和时间流逝产生情感变化。

  • 六种即时情感：孤独感、亲密感、兴奋、烦躁、好奇、忧郁
  • 长期亲密度：随着互动逐渐积累
  • 情感会随时间自然衰减（基于半衰期模型）
  • AI 会在每次对话后主动更新情感状态
  • 情感状态会影响 AI 的回复风格和语气
🔍 性格与关系分析

系统会自动分析对话内容，总结用户性格特征和关系状态。

  • 当对话字符数达到阈值（默认 8000 字符）时自动触发
  • 分析用户的性格特征、情绪倾向、喜好与禁忌
  • 总结当前的关系状态、亲密度变化、关键转折事件
  • 分析结果会注入到后续对话的上下文中
  • 可在 /analysis_status.html 查看分析状态
🤖 主动消息

涟宗也会在你长时间未发消息时主动打招呼。

  • 可在设置中配置触发间隔和空闲时长
  • 支持自定义主动消息的 prompt 模板
  • 可使用变量（如 {user_name}、{idle_minutes}）
  • 可设置是否要求页面可见才触发
  • 使用"立即测试"按钮可以快速测试主动消息
📝 Prompt 注入管理

高级功能：在对话的不同位置注入自定义 prompt。

  • 四种注入位置：system_top（系统提示词开头）、system_bottom（系统提示词结尾）、before_last_user（最后一条用户消息前）、after_last_user（最后一条用户消息后）
  • 支持设置优先级（数字越大越优先）
  • 可以启用/禁用单个注入条目
  • 适合用于添加特定行为指令、角色扮演设定等
🛠️ 工具调用

涟宗也可以调用多种工具来完成任务。

  • open_program: 打开 Windows 程序或文件
  • run_python: 执行 Python 代码
  • read_file / write_file: 读写文件
  • screenshot_and_understand: 截屏并理解屏幕内容
  • read_document: 读取 PDF/EPUB/TXT 文档
  • get_active_window: 获取当前活动窗口
  • list_running_apps: 列出运行中的应用
📖 共读模式

上传书籍文件，与涟宗也一起阅读和讨论。

  • 支持 TXT、PDF、EPUB、Markdown 格式
  • 自动分章节解析
  • 阅读进度自动保存
  • 可以在阅读页面与 AI 讨论书籍内容
🎨 Live2D 模型

支持加载 Live2D 模型，让涟宗也拥有动态形象。

  • 在设置中上传 model3.json 文件
  • 支持眼球追踪、呼吸动画
  • AI 可以通过动作指令控制模型动作和表情
  • 可在设置中启用/禁用各项功能

在主动消息 prompt 和其他可自定义的文本中，可以使用以下变量：

变量名 说明
{user_name} 用户名字（在设置中配置）
{idle_minutes} 用户空闲时长（分钟），仅在主动消息中可用
{time_context} 当前时间段描述（清晨/上午/中午/下午/傍晚/晚上/深夜）
使用示例

主动消息 prompt 示例：

{user_name} 已经 {idle_minutes} 分钟没有回复了，现在是{time_context}。
请主动问候一下，语气要符合涟宗也的性格：冷淡、简短、但隐约透着关心。
